# 仓库说明
本仓库存储了一些我和学校斗智斗勇的结晶。


## 党建平台自动刷课说明
本程序基于Google浏览器编写，需要电脑上安装有Chrome浏览器并进行如下配置
### 1.确认Chrome版本
在Chrome地址栏中输入chrome://version/即可看到浏览器版本
## 2.下载对应版本的Chromediver
去这里：http://npm.taobao.org/mirrors/chromedriver找到对应版本的chromedriver
### 3.将下载的文件放入Chrome的安装路径
### 4.更改为自己的账密
将源代码中对应位置的账密更改为自己的
